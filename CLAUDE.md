## Decision Framework

Before implementing a feature, ask:

1. Does this solve a real business problem?
2. Does it simplify the owner's life?
3. Is there a simpler solution?
4. Is this necessary now?
5. Can this be built incrementally?

If the answer is no, recommend a simpler alternative.
